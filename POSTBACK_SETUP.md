# Postback Setup Guide

This guide explains how to configure the Global Postback URL in your CPA Networks to track leads and revenue in your Dashboard.

## 1. Your Global Postback URL

Your Postback Endpoint is:
```
https://your-domain.com/api/postback.php
```
*(Replace `your-domain.com` with your actual domain or IP)*

## 2. Required Parameters

You must pass the **Click ID** and **Payout** (Revenue) to this endpoint.

| Parameter | Description |
| :--- | :--- |
| `click_id` | **Required**. The unique Click ID generated by the tracker. (Also supports `clickid`, `cid`) |
| `payout` | The revenue amount for the conversion. |
| `currency` | (Optional) Currency code, default is USD. |
| `status` | (Optional) Conversion status, default is 'approved'. |
| `txid` | (Optional) Transaction ID from the network. |

---

## 3. Configuration Examples

Use the following formats for your networks:

### iMonetizeit
```
https://your-domain.com/api/postback.php?network=iMonetizeit&subid=<click_id>&payout=<payout>&country=<country>&os=<traffic_type>
```

### Trafee
```
https://your-domain.com/api/postback.php?network=Trafee&click_id={track}&payout={sum}
```

### Lospollos
```
https://your-domain.com/api/postback.php?network=Lospollos&click_id={cid}&payout={sum}
```

### Clickdealer
```
https://your-domain.com/api/postback.php?network=Clickdealer&click_id=#s2#&payout=#price#
```

---

## 4. How It Works

1. **Traffic**: When a user visits your Smartlink, the tracker generates a unique `clickId`.
2. **Redirect**: The user is redirected to the CPA Network offer with this ID attached (e.g. `&subid=12345`).
3. **Conversion**: When the user converts (signup/sale), the CPA Network fires the **Postback URL**.
4. **Tracking**: The system receives the Postback, matches the ID, and records a **Lead** (+Payout) in your Dashboard database.
5. **Report**: You see the updated **Leads** and **Revenue** in the Reports page.
